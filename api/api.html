API Reference#

The EEGDash API reference curates everything you need to integrate, extend, and automate EEGDashfrom core dataset helpers to feature extraction and rich dataset metadata. The focus is interoperability, extensibility, and ease of use.

The API is organized into three main components:

Core API

Build, query, and manage EEGDash datasets and utilities.

Extract statistical, spectral, and machine-learning-ready features.
